# Salary Predictor using Machine Learning

## 📌 Overview

This project predicts salary using Machine Learning techniques. The dataset is preprocessed using encoding and feature scaling before training a Linear Regression model.

## 🚀 Features

* Data Cleaning
* Missing Value Handling
* One-Hot Encoding
* Label Encoding
* Ordinal Encoding
* Feature Scaling using StandardScaler
* Train-Test Split
* Linear Regression Model
* Correlation Analysis using Heatmap

## 🛠 Technologies Used

* Python
* Pandas
* NumPy
* Matplotlib
* Seaborn
* Scikit-Learn
* Jupyter Notebook

## 📊 Workflow

1. Load Dataset
2. Handle Missing Values
3. Encode Categorical Features
4. Scale Numerical Features
5. Split Data into Training and Testing Sets
6. Train Linear Regression Model
7. Evaluate Model Performance
